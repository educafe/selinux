# SELinux (Security Enhanced Linux)
본 과정은 리눅스 초보자를 포함하여 리눅스 시스템 관리자, 보안담당자가 되기를 희망하는 분을 위한 과정으로
본 과정은 리눅스 시스템 또는 보완 관리자로써 리눅스의 보안을 향상시키는 방법을 학습하는 과정입니다.  
본 과정은 리눅스 기반 제품을 개발하는 엔지니어들이 숙지하여야 할 SELinux Prime 과정이기도 합니다.

### 본 과정에는 다음과 같은 내용을 학습합니다. 
- SELinux와 강제적 접근통제 원칙
- SELinux Context와 Labelling의 기본
- SELinux 하에서의 파일 Labelling과 SELinux 사용자 등록
- SELinux 동작 원칙과 로그정보 확인
- SELinux Policy의 원리와 작성 방법
- SELinux 로그를 활용한 정책 만들기
